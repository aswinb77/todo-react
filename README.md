# 📝 Classic Todo App

A beautiful, retro-style todo application with a classic field theme and smooth animations. Built with React and featuring a unique card-based design with separate sections for active and completed tasks.

## ✨ Features

### 🎨 Design
- **Classic Field Theme** - Warm parchment background with subtle grid pattern
- **Colored Cards** - 5 different warm tones for visual variety
- **Square Cards** - Perfect 1:1 aspect ratio for each todo item
- **Glassmorphism Header** - Subtle blur effect on the input bar
- **Responsive Grid** - Automatically adjusts to screen size

### 📋 Task Management
- **Active Tasks Section** - All pending todos in colorful cards
- **Completed Tasks Section** - Separated below with greenish tones
- **Task Counter** - Shows count for both sections
- **Persistent Storage** - Tasks saved in localStorage

### 🎯 Interactions
- **Completion Circle** - Click to mark tasks as complete
- **2D Disappear Animation** - Tasks perform a flip-and-scale animation when completed
- **Card Appear Animation** - Smooth entrance for new tasks
- **Hover Effects** - Cards lift slightly on hover
- **Edit Functionality** - Prompt-based edit
- **Delete Option** - Remove tasks from either section

### 🎬 Animations
- `cardAppear` - New tasks fade and scale in
- `disappear2D` - Completed tasks do a 2D flip animation
- Smooth transitions on all interactive elements
- Button press effects

## 🛠️ Tech Stack

- **React** - UI library
- **CSS3** - Styling and animations
- **Font Awesome** - Icons
- **Google Fonts** - 'Press Start 2P' & 'Inter' fonts
- **LocalStorage** - Data persistence
